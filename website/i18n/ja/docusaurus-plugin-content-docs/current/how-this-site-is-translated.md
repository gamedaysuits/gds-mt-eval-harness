# このサイトの翻訳について

このサイトは13言語で提供されています。英語以外のすべてのロケールは、
このアリーナと並行して開発された翻訳CLI **Champollion** によって**機械翻訳**されています。各言語の翻訳モデルは**デフォルトではなく、このサイト自身のベンチマーク結果に基づいて選定**されています。各言語ペアは、MT evalハーネスを用いて公開開発コーパス上で評価され、composite scoreが最も高い手法・モデルがそのロケールの翻訳を担当します（統計的に同点の場合はコストで判断）。

読者として知っておくべき点が2つあります。

1. **これらのページは機械翻訳です。** 以下に説明するレジスターおよび用語のガイダンスに従って生成されていますが、すべての文章を人間がレビューしているわけではありません。不自然な表現があった場合、英語版が正式なものとなります。修正のご提案も歓迎します。
2. **選定過程を確認できます。** 以下の各行には、その言語のモデルを選定したベンチマーク実行が記載されており、実行結果は[MT Eval Arenaリーダーボード](https://mtevalarena.org/leaderboard)に公開されています。

## レジスターとトーン

各言語は、Champollionの言語カードから選択された明示的なレジスターで翻訳されており、サイト全体で文体の一貫性が保たれています。

- **Français** — vouvoiement（丁寧な *vous* 形）
- **Deutsch** — Sie-Form
- **Nederlands** — u-vorm
- **Filipino** — 丁寧体、標準的な技術用語を使用
- **Español** — 中立的なラテンアメリカスペイン語
- **简体中文** — 専門的な技術レジスター
- **日本語** — です/ます（丁寧体）
- **한국어** — 해요체（丁寧体）
- **Português** — 専門的なレジスター
- **ไทย** — 中立的な専門体
- **Tiếng Việt** — 中立的な *bạn* 形
- **العربية** — 現代標準アラビア語、専門的なレジスター

## 機械翻訳されないもの

コードブロック、CLIコマンド、設定キー、パッケージ名、URL、および固有名詞は、翻訳時に保護され、設計上英語のまま維持されます。

## 誤訳を見つけた場合

issueまたはPRを作成してください。翻訳済みページのソースはすべて英語の原文です。翻訳済みページへの修正は、英語ソースが変更されていない限り、将来の同期時にも保持されます（同期は英語ソースが変更された場合にのみページを再翻訳します）。
