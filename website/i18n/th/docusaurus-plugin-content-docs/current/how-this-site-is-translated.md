# เว็บไซต์นี้ได้รับการแปลอย่างไร

เว็บไซต์นี้มีให้บริการใน 13 ภาษา ทุก locale ยกเว้นภาษาอังกฤษได้รับการ
**แปลโดยเครื่องผ่าน Champollion** ซึ่งเป็น CLI สำหรับการแปลที่พัฒนาควบคู่กับ
arena นี้ — และโมเดลการแปลสำหรับแต่ละภาษาได้รับการคัดเลือก **โดยเกณฑ์มาตรฐานของเว็บไซต์นี้เอง ไม่ใช่ค่าเริ่มต้น**: คู่ภาษาแต่ละคู่ได้รับการประเมินบน corpus สาธารณะสำหรับการพัฒนาด้วย MT eval harness และวิธีการ/โมเดลที่มี composite score สูงสุด (เมื่อมีความเท่าเทียมทางสถิติจะพิจารณาจากต้นทุน) จะเป็นผู้แปล locale นั้น

นั่นหมายความว่ามีสองสิ่งที่คุณควรทราบในฐานะผู้อ่าน:

1. **หน้าเหล่านี้เป็นการแปลโดยเครื่อง** ผลิตขึ้นพร้อมคำแนะนำด้านรูปแบบภาษาและคำศัพท์ที่อธิบายไว้ด้านล่าง แต่ไม่มีมนุษย์ตรวจสอบทุกประโยค หากมีสิ่งใดอ่านแล้วไม่ถูกต้อง เวอร์ชันภาษาอังกฤษถือเป็นเวอร์ชันที่เชื่อถือได้ — และเรายินดีรับการแก้ไข
2. **คุณสามารถตรวจสอบการคัดเลือกได้** ทุกแถวด้านล่างระบุชื่อ benchmark run ที่คัดเลือกโมเดลสำหรับภาษานั้น โดย run เหล่านั้นได้รับการเผยแพร่ไปยัง [MT Eval Arena leaderboard](https://mtevalarena.org/leaderboard)

`google/gemini-3.1-pro-preview`) ได้รับการให้คะแนนบน corpus สำหรับการพัฒนาของคู่ภาษานั้น ผู้ชนะคือโมเดลที่มี composite score สูงสุด เมื่อโมเดลที่มีต้นทุนต่ำกว่าไม่สามารถแยกแยะได้ทางสถิติจากผู้ทำคะแนนสูงสุด (paired bootstrap resampling, p ≥ 0.05) จะเลือกโมเดลที่มีต้นทุนต่ำกว่า

*Composite score* คือเมตริกคุณภาพแบบผสมของ MT Eval Arena (chrF++,
exact match และ loaded metric plugins ที่ผ่านการตรวจสอบด้วย bootstrap-CI) คะแนนสามารถเปรียบเทียบได้ **ภายในคู่ภาษาเดียวกัน** เท่านั้น ไม่ใช่ข้ามคู่ภาษา — คะแนน 0.28 ในภาษาเกาหลีไม่ได้หมายความว่าหน้าภาษาเกาหลีมีคุณภาพต่ำกว่าหน้าภาษาฝรั่งเศสที่ 0.58 เนื่องจาก corpus และ script แตกต่างกัน

## รูปแบบภาษาและน้ำเสียง

แต่ละภาษาได้รับการแปลด้วยรูปแบบภาษาที่กำหนดไว้อย่างชัดเจนซึ่งคัดเลือกจาก
language cards ของ Champollion เพื่อให้ความเป็นทางการสอดคล้องกันทั่วทั้งเว็บไซต์:

- **Français** — vouvoiement (formal *vous*)
- **Deutsch** — Sie-Form
- **Nederlands** — u-vorm
- **Filipino** — เป็นทางการ พร้อมคำศัพท์ทางเทคนิคมาตรฐาน
- **Español** — ภาษาสเปนละตินอเมริกาแบบกลาง
- **简体中文** — รูปแบบภาษาทางเทคนิคเชิงวิชาชีพ
- **日本語** — です/ます (รูปแบบสุภาพ)
- **한국어** — 해요체 (สุภาพ)
- **Português** — รูปแบบภาษาเชิงวิชาชีพ
- **ไทย** — เป็นกลางและเชิงวิชาชีพ
- **Tiếng Việt** — รูปแบบ *bạn* แบบกลาง
- **العربية** — ภาษาอาหรับมาตรฐานสมัยใหม่ รูปแบบเชิงวิชาชีพ

## สิ่งที่ไม่ได้รับการแปลโดยเครื่อง

Code block, คำสั่ง CLI, configuration key, ชื่อแพ็กเกจ, URL และคำนามเฉพาะได้รับการป้องกันระหว่างการแปลและคงไว้เป็นภาษาอังกฤษตามการออกแบบ

## พบการแปลที่ผิดพลาดหรือไม่?

เปิด issue หรือ PR — ต้นฉบับของทุกหน้าที่แปลแล้วคือต้นฉบับภาษาอังกฤษ การแก้ไขหน้าที่แปลแล้วจะถูกเก็บรักษาไว้ในการซิงค์ครั้งต่อไป ตราบใดที่ต้นฉบับภาษาอังกฤษของหน้านั้นไม่เปลี่ยนแปลง (การซิงค์จะแปลหน้าใหม่เฉพาะเมื่อต้นฉบับภาษาอังกฤษมีการเปลี่ยนแปลง)

*หน้านี้ได้รับการแปลโดยเครื่องด้วยวิธีการที่ระบุในตารางด้านบน — โดยอธิบายการแปลของตัวเองนั่นเอง*
